### Intro

Aven is a Full Stack e-commerce app made using Django, SQLite, and HTML/CSS. 

Features:

- Make listings for items, with images, a description, etc.
- Seperate and filter lisitings into categories
- Bid and comment on listings
- Add to a wishlist and see what listings are on it
- Close listings and find the highest bidder
